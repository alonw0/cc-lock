# @cc-lock/tui

Interactive terminal dashboard for [cc-lock](https://github.com/alonw0/cc-lock), built with [Ink](https://github.com/vadimdemedes/ink) and React.

Launch with:

```bash
cc-lock tui
```

For full documentation see: **https://github.com/alonw0/cc-lock**

## Tabs

| Tab | Key | Description |
|-----|-----|-------------|
| Dashboard | `D` | Live lock status, countdown, quick-lock presets, today's usage |
| Stats | `S` | Bar chart of usage by day/week/month, reset shortcuts |
| Schedules | `C` | Full CRUD for recurring lock schedules |
| Settings | `T` | Toggle boolean settings, edit grace period |
